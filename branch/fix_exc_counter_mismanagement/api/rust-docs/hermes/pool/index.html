Global thread pool for parallel WASM module execution.

This module initializes a global Rayon thread pool for running WASM modules in parallel. The pool uses work-stealing for efficient load balancing across available threads. To avoid saturating the system, the pool leaves a small margin of CPU capacity unused â€” typically 2 threads are reserved for the main application flow and event loop.

Each event may execute multiple WASM modules concurrently within this pool.
